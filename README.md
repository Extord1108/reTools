# reTools

### 介绍

一个用vue.js写的纯前端的标签页网站

### 演示

[exTab-新标签页](https://tab.extord.space/)

### 使用

安装依赖：

```bash
npm install
```

运行：

```bash
npm run server
```



打包：

```bash
npm run build
```
